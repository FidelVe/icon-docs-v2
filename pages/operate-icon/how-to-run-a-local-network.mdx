# How to run a local network

The following documentation describes in detail how to run a local multi-node network for testing purposes using Goloop, the official blockchain architecture implementation of the ICON 2.0 Protocol.

Setting up a multi-node local network requires two general steps:

* [Setting up the local network](/operate-icon/how-to-run-a-local-network/setting-up-a-local-network) using the `gochain-local` repository.
* [Decentralizing the local network](/operate-icon/how-to-run-a-local-network/setting-up-a-local-network) (registering validators, staking and setting up bond for the validators).
* [Debugging the local network](/operate-icon/how-to-run-a-local-network/debugging-a-local-network) (checking the status of the network, checking the status of the validators, and checking the status of the transactions).
